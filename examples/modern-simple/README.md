# Modern Simple Example

A minimal face detection example using WebRTC + face-api.js.

See the **[modern-face-detection-app-example](https://github.com/jeanmachuca/modern-face-detection-app-example)** repository for the complete working implementation.

## What This Example Covers

- WebRTC camera access
- Real-time face detection with TinyFaceDetector
- Drawing bounding boxes on canvas
- FPS counter
- Minimal HTML/CSS/JS (single HTML file)

## How to Run

```bash
npx serve .
# or any static file server
```

Open in a browser that supports WebRTC (Chrome, Firefox, Edge, Safari 14.5+).
